# Промпт для генерации JSON презентаций

Скопируй этот промпт и добавь к своему запросу для DeepSeek, чтобы нейросеть создала презентацию в правильном формате JSON.

---

**Создай презентацию в формате JSON со следующей структурой:**

```json
{
  "slides": [
    {
      "title": "string (обязательно)",
      "sub_title": null или "string (опционально)",
      "content": [
        {
          "type": "text | code | list",
          "content": "string (для text и code)",
          "language": "string (только для type: code, например: javascript, python, html, css, java, cpp, etc.)",
          "items": ["array of strings (только для type: list)"]
        }
      ],
      "left_column": [
        {
          "type": "text | code | list",
          "content": "string (для text и code)",
          "language": "string (только для type: code, например: javascript, python, html, css, java, cpp, etc.)",
          "items": ["array of strings (только для type: list)"]
        }
      ],
      "right_column": [
        {
          // Аналогично left_column
        }
      ]
    }
  ]
}
```

**Правила и требования:**

1. **title** - обязательное поле, заголовок слайда (строка)
2. **sub_title** - опциональное поле, подзаголовок слайда (null или строка)
3. **Расположение контента:**
   - **"content"** - массив элементов контента для отображения БЕЗ колонок (в один столбец). Используй, когда нужно показать контент последовательно, без разделения на колонки
   - **"left_column"** и **"right_column"** - массивы элементов контента для отображения В ДВЕ КОЛОНКИ. Каждый столбец может содержать несколько элементов разных типов
   - **ВАЖНО:** Используй либо `content` (без колонок), либо `left_column`/`right_column` (с колонками). НЕ используй их одновременно!
4. **Типы контента:**
   - **"text"** - обычный текстовый контент. Используй поле `content` со строкой текста
   - **"code"** - блок кода с подсветкой синтаксиса. Используй поля:
     - `type: "code"`
     - `content` - код в виде строки (сохраняй форматирование с \n)
     - `language` - язык программирования (javascript, python, java, html, css, sql, etc.)
   - **"list"** - маркированный список. Используй поле `items` с массивом строк

5. **Важно:**
   - Для типа "text" используй только поле `content`
   - Для типа "code" используй поля `content` и `language` (НЕ используй `items`)
   - Для типа "list" используй только поле `items` (НЕ используй `content`)
   - Каждый слайд должен иметь хотя бы один из вариантов: `content`, `left_column` или `right_column`
   - Столбцы могут быть пустыми массивами, но лучше заполнять их контентом

**Пример правильной структуры:**

**Пример 1: С колонками (left_column и right_column)**
```json
{
  "slides": [
    {
      "title": "Заголовок слайда",
      "sub_title": "Подзаголовок (опционально)",
      "left_column": [
        {
          "type": "text",
          "content": "Это обычный текстовый блок с описанием темы."
        },
        {
          "type": "list",
          "items": [
            "Первый пункт списка",
            "Второй пункт списка",
            "Третий пункт списка"
          ]
        }
      ],
      "right_column": [
        {
          "type": "code",
          "language": "javascript",
          "content": "function example() {\n  console.log('Hello');\n}"
        }
      ]
    }
  ]
}
```

**Пример 2: Без колонок (content)**
```json
{
  "slides": [
    {
      "title": "Заголовок слайда",
      "sub_title": "Подзаголовок (опционально)",
      "content": [
        {
          "type": "text",
          "content": "Это обычный текстовый блок с описанием темы."
        },
        {
          "type": "list",
          "items": [
            "Первый пункт списка",
            "Второй пункт списка",
            "Третий пункт списка"
          ]
        },
        {
          "type": "code",
          "language": "javascript",
          "content": "function example() {\n  console.log('Hello');\n}"
        }
      ]
    }
  ]
}
```

**Создай презентацию по лекции:**

Сделай презентацию информативной, с хорошим балансом текста, списков и примеров кода (если применимо). Количество слайдов должно быть достаточным для раскрытия темы (обычно 5-10 слайдов).

---

## Как использовать:

1. Скопируй весь текст выше (начиная с "Создай презентацию в формате JSON...")
2. Добавь в конце свой запрос с темой презентации, например:
   - "Создай презентацию на тему: Основы React"
   - "Создай презентацию на тему: Алгоритмы и структуры данных"
   - "Создай презентацию на тему: Базы данных SQL"
3. Отправь промпт в DeepSeek
4. Скопируй полученный JSON в новый файл в папке `presentations/`
5. Добавь запись о новой презентации в `presentations/list.json`
